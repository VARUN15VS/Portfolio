## 📂 Featured Projects

Each project below has its own dedicated page on the live site with a custom-built interactive hero, not just a static screenshot.

### 🤖 [CONO-AI](cono.html)
Privacy-first, fully offline voice assistant. Built as part of the Smart India Hackathon–winning team — handles automation, queries, and app control with zero cloud dependency.

### 🎬 [V-Media Player](vmedia.html)
Python & VLC–based media player with dual local/cloud playback, 5GB cloud storage, and JSON-based scene bookmarking to resume exactly where you left off.

### 🛡️ [RulingSafe](RulingSafe.html)
Free, fully offline desktop app for lawyers to manage court rulings, documents, and reference links — built for non-technical users, with zero cloud dependency.

### 📊 [Bullion](bullion.html)
Strategy-driven, parallel stock scanner that applies rule-based technical strategies across an entire watchlist and returns precise buy, stop-loss, and target levels.

### 🗄️ [DB-Saver](db.html)
Python-based backup & restore tool supporting MySQL, PostgreSQL, SQLite, and MongoDB — one consistent interface across SQL and NoSQL engines.

Each project reflects real engineering work — not tutorials.
