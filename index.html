RulingSafe – Offline Legal Ruling & Case Manager

RulingSafe is a desktop-based offline legal management application built to help lawyers and legal professionals organize court rulings, documents, and reference links without relying on Excel files or cloud-based tools.

The application follows a case-centric folder architecture, where each case maintains its own documents, links, metadata, and history in a structured and easily accessible format. Designed specifically for non-technical users, RulingSafe focuses on simplicity, privacy, and reliability.

RulingSafe supports multiple users on the same system, offers fast search and filtering across case numbers, names, courts, and descriptions, and works entirely offline, ensuring that all legal data remains secure and local to the user’s device.

Python PyWebView Desktop Application Offline-First

Bullion - A Multi-Stock Scanner and Trading Assistant

Bullion is a multi-stock trading strategy scanner designed to analyze multiple stocks simultaneously based on user-selected technical strategies. Users can create custom stock lists, select a strategy such as Shooting Star, and run parallel analysis to identify stocks that match the defined conditions.

Built with a strategy-driven and scalable architecture, Bullion evaluates each stock and returns actionable trade levels, including expected buy price, stop-loss, and target sell price. The project demonstrates how software engineering and automation can be applied to financial markets to reduce manual analysis and improve trading consistency.

V-Media Player – Cloud-Connected, Timestamp-Aware Player

V-Media Player is a secure media player supporting both local and online streaming, built using Python (Tkinter, CustomTkinter) and VLC. Each user gets 5GB of cloud storage to stream content from a central server.

Its standout feature is the timestamp system – users can mark, name and revisit their favourite scenes while watching any media. Timestamps are stored locally in JSON, giving users full control and privacy while creating a personal connection to their content.
